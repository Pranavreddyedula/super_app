# 🚀 Super App (Flutter Multi-Utility Application)

A powerful multi-feature Flutter application that combines multiple useful apps into a single platform.

🌐 **Live Demo:**  
https://pranavreddyedula.github.io/super_app/

💻 **GitHub Repository:**  
https://github.com/Pranavreddyedula/super_app

---

## 📱 Features

✨ **Calculator**  
- Perform basic arithmetic operations  
- Simple and responsive UI  

📝 **Notes App**  
- Add and view notes  
- Clean UI for quick usage  

🌦 **Weather App (Live API)**  
- Fetch real-time weather data  
- City-based search  
- Uses OpenWeather API  

✅ **To-Do App**  
- Add daily tasks  
- Lightweight task manager  

---

## 🛠 Tech Stack

- **Flutter** (UI Framework)  
- **Dart** (Programming Language)  
- **REST API** (OpenWeather API)  
- **Git & GitHub** (Version Control)  
- **GitHub Pages** (Deployment)  
- **VS Code** (Development Environment)

## 🚀 Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/Pranavreddyedula/super_app.git
cd super_app
2. Install Dependencies
flutter pub get
3. Run the App
flutter run
4. Build for Web
flutter build web --base-href /super_app/
🔑 API Configuration

This project uses OpenWeather API

Create account at: https://openweathermap.org/

Generate API key

Replace in code:

final apiKey = "YOUR_API_KEY";
